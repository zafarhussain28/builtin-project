# SEGMENT FIT TRAINER

AI-powered physique analysis tool using:
- MediaPipe Pose
- Selfie Segmentation
- Fat score metrics
- Gemini AI recommendations
- Modern UI with custom CSS

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Folder Structure
```
segment-fit-trainer/
│ app.py
│ requirements.txt
│ README.md
│ .gitignore
└── assets/
```

## Notes
- Place your background and logo images inside the `assets/` folder.
- Update the paths in `app.py` accordingly.
